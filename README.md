# Java Battleship Game

Proyecto personal desarrollado durante la asignatura **Programación I** del Grado en Ingeniería Informática.

## Descripción
Juego **Hundir la Flota** implementado en Java para consola.  
Permite colocar barcos y registrar impactos hasta hundir la flota del oponente.  
Este proyecto practica programación básica, uso de matrices y lógica de control en Java.

## Funcionalidades
- Juego por consola con tablero de texto
- Colocación de barcos de manera interactiva
- Registro de impactos y aciertos
- Turnos alternos entre dos jugadores

## Tecnologías
- Java
- Programación orientada a objetos (POO)
- Uso de arrays y estructuras de control
